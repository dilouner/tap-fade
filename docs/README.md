# TapFade - Documentacion del proyecto

Este directorio concentra la documentacion viva del proyecto TapFade. La
intencion es trabajar por modulos: cada modulo se disena, implementa, prueba,
valida y documenta antes de avanzar al siguiente.

## Regla de trabajo

Ningun modulo se considera terminado si no cuenta con:

- Implementacion funcional.
- Pruebas ejecutadas.
- Evidencia de validacion.
- Documentacion actualizada.
- Relacion con requerimientos del FRU.
